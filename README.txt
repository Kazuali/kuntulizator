Кунтулизатор v34 — исправление точного счёта в плей-офф
========================================================

Что изменено:

1. Исправлена логика точного счёта в плей-офф.

   Старое правило v32/v33:
   - точный счёт в плей-офф засчитывался только если участник также угадал команду, которая прошла дальше.

   Новое правило v34:
   - точный счёт считается отдельно от прохода;
   - проход/исход считается отдельно.

2. Пример:

   Факт: Япония — Бразилия 1-1, прошла Бразилия.

   Прогнозы:
   - 1-1 Япония  -> получает только банк точного счёта;
   - 1-1 Бразилия -> получает банк прохода + банк точного счёта;
   - 2-2 Бразилия -> получает только банк прохода;
   - 2-2 Япония -> 0 очков.

3. Apps Script не менялся.
   Ссылка Apps Script сохранена из рабочей версии.

4. Таблица Google Sheets не меняется.
   Формат результатов и прогнозов остаётся прежний:
   - 2-1
   - 1-1 Бразилия
   - 1-1 Япония

Загружать на GitHub нужно только файлы из этого архива.
Apps Script на GitHub не загружать.
