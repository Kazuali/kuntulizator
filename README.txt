Кунтулизатор — статический сайт MVP

Что внутри:
- index.html — страница сайта
- styles.css — стили
- data.js — данные: участники, матчи, прогнозы, будущие результаты
- app.js — логика подсчёта очков и отображения таблиц

Как открыть локально:
1. Распакуйте архив.
2. Откройте index.html в браузере.

Как вносить фактический результат:
1. Откройте data.js.
2. Найдите нужный матч в блоке matches.
3. Замените:
   "homeScore": null,
   "awayScore": null
   на, например:
   "homeScore": 2,
   "awayScore": 1
4. Сохраните файл и обновите страницу.

Правила:
- 1400 очков делятся между всеми, кто угадал исход.
- 700 очков делятся между всеми, кто угадал точный счёт.
- Если победителей по соответствующему банку нет, банк сгорает.
- Округление — до 2 знаков.

Важное замечание:
В исходной таблице часть прогнозов вида 2-1 была автоматически превращена Excel/Google Sheets в даты. В MVP они восстановлены как счёт по логике день-месяц: 2 января = 2-1, 5 февраля = 5-2 и т.д.


Мобильная версия:
Сайт адаптирован под телефон: блоки становятся в один столбец, меню прокручивается горизонтально, таблицы можно двигать пальцем влево-вправо.
