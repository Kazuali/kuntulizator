Кунтулизатор — версия с автообновлением из Google Sheets

Что изменилось:
1. Сайт читает данные из опубликованной CSV-ссылки Google Sheets.
2. Обновление происходит автоматически каждые 60 секунд.
3. Есть кнопка «Обновить данные» для ручной проверки.
4. Результаты нужно вводить в строку «РЕЗУЛЬТАТ» под соответствующим матчем.

CSV-ссылка:
https://docs.google.com/spreadsheets/d/e/2PACX-1vSpyPbcc20eeE0khQNgK5xBDvhUmJMdLClNhl0YgH31GMriyF2-yeMyw1WNzENB7efQZUP-5PNeSAtd/pub?gid=0&single=true&output=csv

Формат результата:
2-1
1-0
0-0
3-2

Важно:
- Не меняйте строку с названием «Участники / Матч».
- Не меняйте название строки «РЕЗУЛЬТАТ».
- Не объединяйте ячейки внутри блока матчей/прогнозов/результатов.
- Не добавляйте телефоны, адреса и другие личные данные в опубликованный лист.

Как обновить сайт на GitHub:
1. Распакуйте архив.
2. Откройте папку kuntulizator_site_autoupdate.
3. Загрузите файлы index.html, styles.css, data.js, app.js, README.txt на GitHub поверх старых.
4. Нажмите Commit changes.
5. Через 1-3 минуты сайт обновится.
