Кунтулизатор — версия со статистикой и чистым интерфейсом

Что есть в версии:
- данные автоматически читаются из опубликованной Google Sheets CSV-ссылки;
- автообновление каждые 60 секунд;
- кнопка «Обновить» для ручного обновления;
- лишние технические подписи на сайте убраны;
- добавлена вкладка «Статистика»;
- статистика считается на стороне сайта автоматически.

Как обновить сайт на GitHub:
1. Распакуйте архив.
2. Откройте папку kuntulizator_site_stats_clean.
3. Загрузите все файлы изнутри папки в корень репозитория GitHub.
4. Нажмите Commit changes.
5. Подождите 1–3 минуты и откройте сайт.

Как заполнять результаты:
- в Google Sheets в строке «РЕЗУЛЬТАТ» под нужным матчем введите счёт в формате 2-1;
- сайт сам пересчитает таблицу и статистику;
- если нужно ускорить обновление, нажмите кнопку «Обновить» на сайте.

Что не менять без необходимости:
- строку с названиями участников;
- строку «РЕЗУЛЬТАТ»;
- названия матчей в строке заголовков;
- порядок участников и матчей.
